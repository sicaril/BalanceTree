# BalanceTree

本项目为平衡树的性能对比C++测试工程，其中包含了：

1. AVL 基于节点高度差的增删逻辑，包含递归和非递归实现。
2. AVL 基于节点平衡因子的增删逻辑，包含递归和非递归实现。
3. RBT 红黑树的增删逻辑，包含多种递归和非递归的实现。
4. 各平衡树的校验逻辑，及性能测试。

对应的视频教程：https://space.bilibili.com/1700469421
